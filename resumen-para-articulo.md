# Nota sugerida para el artículo

El EAN-IN fue evaluado mediante modelos de Teoría de Respuesta al ítem para ocho constructos: Planteamiento del problema, Revisión de la literatura, Metodología, Método, Resultados, Discusión, Conclusiones y Comunicación científica. Como material suplementario, se habilitó un micrositio en GitHub Pages que presenta la evidencia psicométrica completa por constructo, incluyendo unidimensionalidad/ajuste global, independencia local, umbrales, cargas factoriales, coeficientes, ajuste de ítems, fiabilidad marginal, diccionario de reactivos y figuras. El sitio no publica datos crudos ni respuestas individuales.
